# Public Health Analytics — SAS · Python · R

*By Aly Drame, MD, MPH, MBA.* One public-health data-analysis curriculum, taught three times over in the three languages an epidemiologist actually meets on the job. Each language repo follows the **same 12-topic workflow** on the **same datasets**, so you can pick the language you need — or read them side by side to see how the same task looks in each.

## The three language tracks

| Track | Repository | Stack |
|---|---|---|
| SAS | [sas-public-health-analytics](https://github.com/Drame-social/sas-public-health-analytics) | Base SAS, PROC steps, macros, ODS |
| Python | [Introduction-to-Python](https://github.com/Drame-social/Introduction-to-Python) | pandas, numpy, matplotlib, seaborn |
| R | [R-Practice](https://github.com/Drame-social/R-Practice) | tidyverse, ggplot2, lubridate |

The titles are deliberately parallel — *Public Health Analytics — SAS / Python / R* — because the content is the same; only the software differs.

## Shared workflow

All three tracks move through the same twelve topics, in the same order:

```
01 Environment setup
02 Data import & creation
03 Data merging & subsetting
04 Categorical variables & dates
05 Conditionals & type conversion
06 Functions
07 Loops / vectorization / arrays
08 Longitudinal & repeated measures
09 Variable documentation
10 Reporting & export
11 Automation (macros / pipelines / functions)
12 Visualization & mapping
```

## Data

All three tracks use the same educational public-health datasets from **AEPI537D (Introduction to SAS Programming), Rollins School of Public Health, Emory University** — synthetic or course-provided, with no real patient or program records. This keeps the three tracks directly comparable and avoids any confidential data.

## How to use this

- **Learning one language?** Go straight to that track and work topics 01 → 12.
- **Moving between languages?** Open two tracks side by side on the same topic number to see the direct translation.
- **Reviewing the portfolio?** Each track stands alone as a complete worked curriculum; together they show the same analyses implemented three ways.
